Items

Items are the basic building blocks of your game. They can be used to create weapons, armor, and other equipment.

To create an item, you need to define its properties such as name, type, and attributes.

Creating an Item

Use the following syntax to create an item:

item [name] [type] [attributes]

Example

item Sword Weapon { damage: 10, durability: 100 }

This creates a sword with a damage of 10 and a durability of 100.
